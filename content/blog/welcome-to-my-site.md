This is the first post on my new personal site. The whole thing — including
this blog — is generated by a small static site generator I wrote in Rust.

## How it works

- Content lives in `content/` as Markdown files with TOML front matter
- A Rust program (`src/main.rs`) parses the Markdown with `pulldown-cmark`
  and renders pages using `tera` templates
- The output is plain HTML/CSS in `dist/`, ready to deploy anywhere static
  files are served — GitHub Pages, Netlify, S3, you name it
